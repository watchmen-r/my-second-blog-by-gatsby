Event loop

Event loop is not defined by ECMA script. So this should be implemented by environment like a browser or runtime( node, demo). There would be able to exist multiple task queue. For example, blink, which is a rendering engine in chrome, prioritize to process click event.

JavaScript is a single thread programming language, and this means, program is executed by single thread and have single call stack. If RuntimeAPI like fetch() blocks the process, main process would have to wait until fetch() is finished. So to avoid these situations, Javascript engine doesnâ€™t involve to RuntimeAPI such a fetch(), and when the fetch() gets the data, notice to JavaScript engine with data to register the callback function.
